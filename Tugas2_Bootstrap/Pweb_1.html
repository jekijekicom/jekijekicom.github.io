Apa itu Web ?

World Wide Web atau disingkat 'www' adalah sekumpulan webpages dari berbagai belahan dunia yang dapat ditemukan dalam internet. Awal terbentuknya web pada tahun 1989 oleh Sir Tim Berners-Lee, ditujukan untuk para ilmuwan sebagai media untuk menyebarkan hasil penemuan mereka dengan mudah. Sekarang, tidak hanya para ilmuwan saja yang memakai web, semua orang menggunakannya dengan tujuan masing-masing.
Terdapat beberapa bahasa yang sering digunakan dalam memrogram web, antara lain:

HTML

Hypertext Markup Language adalah bahasa pemrograman untuk merancang konten apa saja yang akan ditampilkan pada web yang dibuat.

CSS

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemograman.

Javascript

Bahasa untuk memrogram operasi atau behavior yang ada pada sebuah web.

Dalam bidang web development, terbagi beberapa kelompok sesuai dengan perannya masing-masing, antara lain:

UI/UX Designers

bertugas sebagai pembuat design tampilan website, kalo designnya sudah disetujui oleh client maka designnya akan diberikan ke bagian frontend untuk selanjutnya dibuatkan tampilan websitenya.

Front-end

mengimplementasikan design yang sudah dibuat ke dalam codingan berupa html, css dan javascript.

Back-end

bertugas menghubungkan tiap halaman yang sudah dibuat oleh bagian front-end, membuat database dan menghubungkannya kedalam website, menampilkan data ke dalam website sesuai dengan data yang dibutuhkan pada tiap-tiap halaman.

Info!Bagian yang disebut di atas merupakan yang paling umum dan web berskala kecil. Ada banyak lagi bagian sesuai dengan kebutuhan web itu sendiri.
